# <a name="bass-boost"></a>低音增强


在具有有限低音功能的扬声器的系统（如便携式计算机）中，有时可以通过在扬声器支持的频率范围内提升低音响应来提高音频的感知质量。 通过提高中端低音范围内的增益，低音提高了移动设备上非常小的扬声器声音。
