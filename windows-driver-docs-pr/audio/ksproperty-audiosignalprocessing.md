# <a name="ksproperty_audiosignalprocessing-enumeration"></a>KSPROPERTY \_ AUDIOSIGNALPROCESSING 枚举


KSPROPERTY \_ AUDIOSIGNALPROCESSING 枚举定义音频驱动程序在与 pin 上音频处理模式连接时使用的常量。

<a name="syntax"></a>语法
------

```ManagedCPlusPlus
typedef enum _KSPROPERTY_AUDIOSIGNALPROCESSING { 
  KSPROPERTY_AUDIOSIGNALPROCESSING_MODES
} KSPROPERTY_AUDIOSIGNALPROCESSING;
```

<a name="constants"></a>常量
---------

<span id="KSPROPERTY_AUDIOSIGNALPROCESSING_MODES"></span><span id="ksproperty_audiosignalprocessing_modes"></span>**KSPROPERTY \_ AUDIOSIGNALPROCESSING \_ 模式**  
指定 [**KSPROPERTY \_ AUDIOSIGNALPROCESSING \_ 模式**](ksproperty-audiosignalprocessing-modes.md) 属性的 ID。
