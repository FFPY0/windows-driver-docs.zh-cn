# <a name="ksproperty_rtaudio_query_notification_support"></a>KSPROPERTY \_ RTAUDIO \_ 查询 \_ 通知 \_ 支持


客户端应用程序使用 `KSPROPERTY_RTAUDIO_QUERY_NOTIFICATION_SUPPORT` 属性来确定音频驱动程序是否可以在提交的缓冲区上执行的进程完成时通知客户端应用程序。
