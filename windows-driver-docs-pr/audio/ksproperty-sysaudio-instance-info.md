# <a name="ksproperty_sysaudio_instance_info"></a>KSPROPERTY \_ SYSAUDIO \_ 实例 \_ 信息

没有为此属性定义 (操作数据) 的属性值。 将属性值指针指定为 **NULL**，将属性值大小指定为零。
