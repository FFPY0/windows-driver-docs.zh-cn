# <a name="low-frequency-protection"></a>低频保护


低频率保护是一种 [转发低音管理](bass-management.md)的形式。 它在没有低音炮并且所有扬声器都很小且缺少低音功能时使用。 当你应用低频率保护 APO 时，结果是删除音频信号的部分，该部分低于 APO 算法指定的截止时间间隔。
