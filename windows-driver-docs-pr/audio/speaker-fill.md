# <a name="speaker-fill"></a>扬声器填充


大多数音乐仅具有两个通道，因此不会针对典型音频或视频发烧的多通道音频设备进行优化。 因此，只需使用使用和 loudspeakers 的音乐，就不会获得理想的音频体验。 演讲者填充模拟多通道扬声器设置。 它允许在房间内的所有 loudspeakers 上播放仅在两个扬声器上播放的音乐，从而增强空间成名。
