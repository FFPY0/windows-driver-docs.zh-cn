# <a name="wdm-audio-components"></a>WDM 音频组件


## <span id="wdm_audio_components"></span><span id="WDM_AUDIO_COMPONENTS"></span>


本部分介绍系统提供的 Microsoft Windows 驱动模型 (WDM) 音频组件。 这些是操作系统附带的软件组件。 这些组件支持 ISA、PCI 和 USB 音频设备，并提供应用程序用于控制和监视这些设备的软件接口。 本文讨论了以下主题：

[用户模式 WDM 音频组件](user-mode-wdm-audio-components.md)

[内核模式 WDM 音频组件](kernel-mode-wdm-audio-components.md)

[Wave 和 DirectSound 组件](wave-and-directsound-components.md)

[MIDI 和 DirectMusic 组件](midi-and-directmusic-components.md)

[虚拟音频设备](virtual-audio-devices.md)
