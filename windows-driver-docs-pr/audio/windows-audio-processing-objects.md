音频处理对象 () ，为 Windows 音频流提供可自定义的基于软件的数字信号处理。

本部分包含以下主题。

- [音频处理对象体系结构](audio-processing-object-architecture.md)
- [实现音频处理对象](implementing-audio-processing-objects.md)
