# <a name="-command-help"></a>? （命令帮助）


问号 (**？**) 字符将显示所有命令和运算符的列表。

**注意**   问号本身显示命令帮助。 [**？表达式**](---evaluate-expression-.md)语法计算给定的表达式。

```dbgcmd
    ?
```
