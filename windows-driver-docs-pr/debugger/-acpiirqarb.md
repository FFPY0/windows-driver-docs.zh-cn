# <a name="acpiirqarb"></a>!acpiirqarb


**！ Acpiirqarb** EXTENSION (ACPI) IRQ 仲裁器结构中显示高级配置和电源接口的内容，该结构包含将 i/o 设备到系统中断控制器输入的配置，以及处理器中断调度表 (IDT) 条目。

```dbgcmd
!acpiirqarb
```
