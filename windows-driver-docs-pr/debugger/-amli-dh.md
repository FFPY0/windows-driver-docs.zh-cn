# <a name="amli-dh"></a>!amli dh


**！ Amli dh** 扩展显示 AML 解释器的内部堆块。

语法

```dbgcmd
!amli dh [HeapAddress]
```

## <a name="span-idddk__amli_dh_dbgspanspan-idddk__amli_dh_dbgspanparameters"></a><span id="ddk__amli_dh_dbg"></span><span id="DDK__AMLI_DH_DBG"></span>参数


<span id="_______HeapAddress______"></span><span id="_______heapaddress______"></span><span id="_______HEAPADDRESS______"></span>*HeapAddress*   
指定堆块的地址。 如果省略此，则显示全局堆。
