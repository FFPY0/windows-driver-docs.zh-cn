# <a name="btb"></a>!btb


**！ Btb** extension 显示当前处理器的基于 Itanium 的处理器、分支跟踪缓冲区 (btb) 配置和跟踪寄存器。

```dbgcmd
!btb
```

**重要提示**  此命令在 Windows 调试器版本10.0.14257 和更高版本中已弃用，不再可用。
