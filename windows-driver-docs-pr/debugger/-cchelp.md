# <a name="cchelp"></a>!cchelp


对于某些缓存管理扩展， **！ cchelp** 扩展在调试器命令窗口中显示一些简短帮助文本。

```dbgsyntax
!cchelp
```

**！ Cchelp** extension 显示 [**！ bcb**](-bcb.md)， [**！ defwrites**](-defwrites.md)， [**！ finddata**](-finddata.md)， [**！ scm**](-scm.md)缓存管理扩展的帮助。 其他缓存管理扩展包括 [**！ openmaps**](-openmaps.md) 和 [**！ pcm**](-pcm.md)。
