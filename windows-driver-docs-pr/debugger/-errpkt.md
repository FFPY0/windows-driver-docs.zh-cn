# <a name="errpkt"></a>!errpkt


**！ Errpkt** 扩展显示 Windows 硬件错误体系结构 (WHEA) 硬件错误数据包的内容。

```dbgcmd
!errpkt Address 
```

## <a name="span-idddk__ubp_dbgspanspan-idddk__ubp_dbgspanparameters"></a><span id="ddk__ubp_dbg"></span><span id="DDK__UBP_DBG"></span>参数

指定硬件错误数据包的地址。
