# <a name="pars"></a>!pars


**！ Pars** 扩展显示指定的处理器应用程序注册文件。

```dbgcmd
!pars Address
```

**重要提示**  此命令在 Windows 调试器版本10.0.14257 和更高版本中已弃用，不再可用。
