# <a name="rcdrkdrcdrsearchpath"></a>!rcdrkd.rcdrsearchpath


**！ Rcdrkd rcdrsearchpath** (TMF) 和跟踪消息控件 (TMC) 文件中的跟踪消息格式设置搜索路径。

```dbgcmd
!rcdrkd.rcdrsearchpath FilePath
```
