# <a name="rrestart-register-for-restart"></a>.rrestart（注册以重启）


如果重新启动或应用程序发生故障，则 **rrestart** 命令将注册调试会话以进行重新启动。

```dbgcmd
.rrestart
```

<a name="remarks"></a>备注
-------

此命令不适用于提升的调试器会话。
