# <a name="urestart-unregister-for-restart"></a>.urestart（取消注册以重启）


如果重新启动或应用程序发生故障，则 **urestart** 命令将取消注册调试会话以进行重新启动。

```dbgcmd
.urestart
```

<a name="remarks"></a>备注
-------

此命令不适用于提升的调试器会话。
