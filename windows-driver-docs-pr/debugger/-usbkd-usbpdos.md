# <a name="usbkdusbpdos"></a>!usbkd.usbpdos


**！ Usbkd usbpdos** 命令显示有关 USB 集线器驱动程序创建 (PDOs) 的所有物理设备对象的信息。

```dbgcmd
!usbkd.usbpdos
```

## <a name="span-iddllspanspan-iddllspandll"></a><span id="DLL"></span><span id="dll"></span>.DLL


Usbkd.dll
