# <a name="wmitracehelp"></a>!wmitrace.help


**！ Wmitrace** 在 Wmitrace.dll 中显示扩展命令。

```dbgcmd
!wmitrace.help 
```

## <span id="ddk__wmitrace_help_dbg"></span><span id="DDK__WMITRACE_HELP_DBG"></span>


### <a name="span-iddllspanspan-iddllspandll"></a><span id="DLL"></span><span id="dll"></span>.DLL

此扩展由 Wmitrace.dll 导出。

此扩展在 windows 2000 和更高版本的 Windows 中可用。 如果要将此扩展与 Windows 2000 一起使用，必须先将 Wmitrace.dll 文件从 Windows 的调试工具安装目录的 winxp 子目录复制到 w2kfre 子目录。
