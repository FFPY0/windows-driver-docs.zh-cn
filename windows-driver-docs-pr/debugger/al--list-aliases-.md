# <a name="al-list-aliases"></a>al（列出别名）


**Al** 命令显示所有当前定义的用户命名别名的列表。

```dbgcmd
al
```

<a name="remarks"></a>备注
-------

**Al** 命令列出所有用户命名的别名。 但此命令不会列出 ($u 0 到 $u 9) 的固定名称别名。
