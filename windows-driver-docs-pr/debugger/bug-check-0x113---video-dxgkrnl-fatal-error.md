# <a name="bug-check-0x113-video_dxgkrnl_fatal_error"></a>Bug 检查0x113：视频 \_ DXGKRNL \_ 严重 \_ 错误


视频 \_ DXGKRNL \_ \_ 错误 bug 检查的值为0x00000113。 这表明 Microsoft DirectX 图形内核子系统已检测到冲突。

此 bug 检查很少出现。

> [!IMPORTANT]
> 本主题面向程序员。 如果您是在使用计算机时收到蓝屏错误代码的客户，请参阅[蓝屏错误疑难解答](https://www.windows.com/stopcode)。
