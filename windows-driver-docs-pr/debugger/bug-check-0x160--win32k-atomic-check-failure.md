# <a name="bug-check-0x160-win32k_atomic_check_failure"></a>Bug 检查0x160： WIN32K.SYS \_ 原子 \_ 检查 \_ 失败


WIN32K.SYS \_ 原子 \_ 检查 \_ 失败 bug 检查的值为0x00000160。 这表示 Win32k.sys 函数违反了 ATOMICCHECK。

> [!IMPORTANT]
> 本主题面向程序员。 如果您是在使用计算机时收到蓝屏错误代码的客户，请参阅[蓝屏错误疑难解答](https://www.windows.com/stopcode)。


## <a name="win32k_atomic_check_failure-parameters"></a>WIN32K.SYS \_ 原子 \_ 检查 \_ 失败参数


| 参数 | 描述                                                             |
|-----------|-------------------------------------------------------------------------|
| 1         | 当前在原子操作内的堆栈上的函数计数 |
| 2         | 预留                                                                |
| 3         | 预留                                                                |
| 4         | 预留                                                                |
