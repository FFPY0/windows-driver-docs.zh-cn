# <a name="bug-check-0x163-worker_thread_test_condition"></a>Bug 检查0x163：工作 \_ 线程 \_ 测试 \_ 条件


工作 \_ 线程 \_ 测试 \_ 条件 bug 检查的值为0x00000163。 这表示内核工作线程的测试引发了故障。

> [!IMPORTANT]
> 本主题面向程序员。 如果您是在使用计算机时收到蓝屏错误代码的客户，请参阅[蓝屏错误疑难解答](https://www.windows.com/stopcode)。


## <a name="worker_thread_test_condition-parameters"></a>工作 \_ 线程 \_ 测试 \_ 条件参数


| 参数 | 描述                                               |
|-----------|-----------------------------------------------------------|
| 1         | 活动测试标志                                         |
| 2         | 对应于触发失败的测试的标志 |
| 3         | 预留                                                  |
| 4         | 预留                                                  |
