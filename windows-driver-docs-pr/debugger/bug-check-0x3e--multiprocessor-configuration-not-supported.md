# <a name="bug-check-0x3e-multiprocessor_configuration_not_supported"></a>Bug 检查0x3E： \_ \_ 不 \_ 支持多处理器配置


不支持的多处理器 \_ 配置 \_ \_ bug 检查的值为0x0000003E。 这表明系统具有多个处理器，但彼此之间不对称。

> [!IMPORTANT]
> 本主题面向程序员。 如果您是在使用计算机时收到蓝屏错误代码的客户，请参阅[蓝屏错误疑难解答](https://www.windows.com/stopcode)。


## <a name="multiprocessor_configuration_not_supported-parameters"></a>多处理器 \_ 配置 \_ 不支持的 \_ 参数


无

<a name="cause"></a>原因
-----

为了进行对称，所有处理器必须具有相同的类型和级别。 此系统包含不同类型的处理器 (例如，奔腾处理器和80486处理器) 。
