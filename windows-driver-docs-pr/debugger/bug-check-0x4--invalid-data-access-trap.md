# <a name="bug-check-0x4-invalid_data_access_trap"></a>Bug 检查0x4：无效的 \_ 数据 \_ 访问 \_ 陷阱


无效的 \_ 数据 \_ 访问 \_ 陷阱错误检查的值为0x00000004。 它表示数据访问陷阱不正确。

此 bug 检查很少出现。

> 本主题面向程序员。 如果您是在使用计算机时收到蓝屏错误代码的客户，请参阅[蓝屏错误疑难解答](https://www.windows.com/stopcode)。
