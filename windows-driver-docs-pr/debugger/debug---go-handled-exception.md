# <a name="debug--go-handled-exception"></a>调试 | 转到已处理的异常


## <span id="ddk_debug_go_handled_exception_dbg"></span><span id="DDK_DEBUG_GO_HANDLED_EXCEPTION_DBG"></span>


单击 "**调试**" 菜单上的 "**跳过处理的异常**" 可恢复目标上的执行，并将当前异常视为已处理。
