# <a name="debug--go-unhandled-exception"></a>调试 | 转到未经处理的异常


## <span id="ddk_debug_go_unhandled_exception_dbg"></span><span id="DDK_DEBUG_GO_UNHANDLED_EXCEPTION_DBG"></span>


单击 "**调试**" 菜单上的 "退出 **未经处理的异常**" 可恢复目标上的执行，并将当前异常视为未处理的异常。
