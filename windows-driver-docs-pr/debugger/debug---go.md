# <a name="debug--go"></a>调试 | 转到


## <span id="ddk_debug_go_dbg"></span><span id="DDK_DEBUG_GO_DBG"></span>


单击 "**调试**" 菜单上的 **"开始"** 以继续 (或开始) 目标上执行。 此执行将继续，直到到达断点、发生异常或事件、进程结束或调试器进入目标。

此命令等效于按 F5，或单击 **Go (F5)** ![ ](images/tbgo.png) 工具栏上的 "开始") 按钮 (屏幕截图中的 " (") "按钮。
