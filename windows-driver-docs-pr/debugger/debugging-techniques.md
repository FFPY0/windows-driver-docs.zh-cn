# <a name="debugging-techniques"></a>调试方法


本部分介绍两种类型的调试技术：标准和专用化。 标准技术适用于大多数调试方案，例如，设置断点、检查调用堆栈和查找内存泄漏。 专用技术适用于特定技术或代码类型，示例即插即用调试、内核模式驱动程序框架调试和 RPC 调试。
