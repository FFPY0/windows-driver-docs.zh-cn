# <a name="ending-a-debugging-session-in-visual-studio"></a>在 Visual Studio 中结束调试会话

> [!IMPORTANT]
> 此功能在 Windows 10 版本1507及更高版本的 WDK 中不可用。
>

若要在 Microsoft Visual Studio 中结束调试会话，请在 " **调试** " 菜单中选择 " **停止调试**"。
