# <a name="file--delete-workspaces"></a>文件 | 删除工作区


## <span id="ddk_file_delete_workspaces_dbg"></span><span id="DDK_FILE_DELETE_WORKSPACES_DBG"></span>


单击 "**文件**" 菜单上的 "**删除工作区**" 以删除一个或多个现有工作区。

### <a name="span-iddialog_boxspanspan-iddialog_boxspandialog-box"></a><span id="dialog_box"></span><span id="DIALOG_BOX"></span>对话框

单击 " **删除工作区**" 时，将显示 " **删除工作区** " 对话框。 在此对话框中，选择要删除的工作区，然后单击 " **删除**"。

单击 **“关闭”** 以关闭对话框。

### <a name="span-idadditional_informationspanspan-idadditional_informationspanadditional-information"></a><span id="additional_information"></span><span id="ADDITIONAL_INFORMATION"></span>附加信息

有关不同级别的工作区以及如何使用它们的详细信息，请参阅 [使用工作区](using-workspaces.md)。
