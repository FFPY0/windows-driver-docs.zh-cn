# <a name="source-control-systems"></a>源代码管理系统


Srcsrv.ini 包括 Visual SourceSafe (vss.pm) 、Perforce (p4.pm) 、Team Foundation Server (tfn.pm) 和 Subversion (svn.pm) 的提供程序模块。 还有一个并发版本系统 (CVS) 模块，可用于 Srcsrv.ini，但尚未广泛测试。 还可以生成您自己的模块以支持其他版本的控制系统。
