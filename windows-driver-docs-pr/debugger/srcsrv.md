# <a name="srcsrv"></a>SrcSrv


Srcsrv.ini 工具 ( # A0) 使客户端能够检索用于生成应用程序的源文件的确切版本。 由于模块的源代码可以在不同的版本之间更改，因此，在生成相关模块的版本时，查看源代码是非常重要的。

Srcsrv.ini 从源代码管理中检索相应的文件。 若要使用 Srcsrv.ini，必须对应用程序进行源索引。
