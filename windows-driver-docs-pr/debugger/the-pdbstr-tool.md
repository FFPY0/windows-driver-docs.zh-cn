# <a name="the-pdbstr-tool"></a>PDBStr 工具


索引脚本使用 Pdbstr.exe ( # A0) 工具将版本控制信息插入目标 .pdb 文件的 "srcsrv.ini" 备用流。 它还可以从 .pdb 文件中读取任何流。 您可以使用此信息来验证索引脚本是否正常工作。

有关详细信息，请运行包含 **-？** 的 pdbstr.exe 选项。
