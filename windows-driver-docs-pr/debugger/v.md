# <a name="v"></a>V


<span id="virtual_process"></span><span id="VIRTUAL_PROCESS"></span>**虚拟进程**  
在内核模式调试中，调试器引擎创建单个 *虚拟进程* 来表示目标的内核。

<span id="virtual_thread"></span><span id="VIRTUAL_THREAD"></span>**虚拟线程**  
在内核模式调试中，调试器引擎为目标计算机中的每个处理器创建一个 *虚拟线程* 。
