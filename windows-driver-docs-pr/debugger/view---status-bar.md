# <a name="view--status-bar"></a>视图 | 状态栏


## <span id="ddk_view_status_bar_dbg"></span><span id="DDK_VIEW_STATUS_BAR_DBG"></span>


选择或清除 "**视图**" 菜单上的 **状态栏**，使状态栏变为可见或不可见。

有关如何使用状态栏的详细信息，请参阅 [使用工具栏和状态栏](using-the-toolbar-and-status-bar.md)。
