# <a name="window--close-all-error-windows"></a>窗口 | 关闭所有错误窗口


## <span id="ddk_window_close_all_error_windows_dbg"></span><span id="DDK_WINDOW_CLOSE_ALL_ERROR_WINDOWS_DBG"></span>


单击 "**窗口**" 菜单上的 "**关闭所有错误窗口**" 以关闭从找不到源文件中打开的所有错误消息框。
