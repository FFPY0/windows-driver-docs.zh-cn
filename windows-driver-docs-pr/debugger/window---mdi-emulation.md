# <a name="window--mdi-emulation"></a>窗口 | MDI 模拟


选择 "**窗口**" 菜单上的 " **MDI 模拟**" 以使 WinDbg 模拟多文档界面， (MDI) 的窗口样式。 此类窗口不同于停靠模式，因为所有窗口都是浮动窗口，但浮动窗口在框架窗口内受到限制。 此行为在引入停靠模式之前模拟 WinDbg 的行为。

清除 "**窗口**" 菜单上的 **MDI 模拟**，以将 WinDbg 返回到停靠模式。
