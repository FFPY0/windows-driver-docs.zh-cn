# <a name="c28158"></a>C28158


警告 C28158：未保存 IRQL

**\_ IRQL \_ 保存 \_** 批注指定当前函数会将 irql 值保存在指定的变量中，但至少有一个路径中的 irql 值未保存在该变量中。
