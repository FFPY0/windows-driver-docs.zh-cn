# <a name="c28268"></a>C28268


警告 C28268：函数的函数类与此处使用的 typedef 上的函数类不匹配

此警告意味着函数类上的批注与类型声明指定的函数类型不匹配。 此警告表明批注而不是所分析的代码中存在错误。
