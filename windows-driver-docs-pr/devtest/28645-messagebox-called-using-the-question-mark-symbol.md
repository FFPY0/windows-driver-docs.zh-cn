# <a name="c28645"></a>C28645


警告 C28645：使用不再推荐的问号消息符号调用了 MessageBox

不再建议在模式对话框中使用问号消息图标，因为它并不清楚地表示特定类型的消息，而作为问题的消息短语可能适用于任何消息类型。 此外，用户可能会将问号消息符号与帮助信息混淆。

因此，不要在消息框中使用问号消息符号。 系统继续支持它包含的内容，只为满足反向兼容性。
