# <a name="c28730"></a>C28730


警告 C28730：可能会 \\ \\ 直接将 "0" 分配给指针。

此警告表明可能出现打字错误：已将 **nul** 字符赋给指针;这可能是因为字符应作为字符串终止符，并应分配给指针指向的内存。
