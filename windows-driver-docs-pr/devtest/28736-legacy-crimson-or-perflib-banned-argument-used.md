# <a name="c28736"></a>C28736

警告 C28736：已禁止的 API 参数用法

此警告意味着使用了旧的事件日志或其他跟踪 API，或使用了禁止参数的 Perflib 使用者 API。 这些 Api 和参数已替换为 Windows 模型的事件跟踪，这可提高性能和安全性。
