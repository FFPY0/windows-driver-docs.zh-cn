是的。 使用标准过程将软件跟踪添加到驱动程序的源代码。 有关详细信息，请参阅 [在内核模式驱动程序中初始化 WPP 软件跟踪如何实现？](how-do-i-initialize-wpp-software-tracing-in-a-kernel-mode-driver-.md)。
