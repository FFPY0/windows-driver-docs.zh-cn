# <a name="changing-the-text-size-in-the-defect-viewer"></a>更改缺陷查看器中的文本大小


若要更改 "缺陷查看器" 窗口中的文本大小，请在 " **视图** " 菜单中选择 " **更小** 或 **更大**"，或按 F1 (较小) 或 F2 (更大的) 键。 您可以重复按它们来增加或减少文本大小，直到它达到最小值或最大大小。

此功能仅影响缺陷查看器。 它不会更改 "结果" 窗格中的文本大小。
