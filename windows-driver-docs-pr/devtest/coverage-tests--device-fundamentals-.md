# <a name="coverage-tests-device-fundamentals"></a>覆盖范围测试（设备基础功能）


设备基础覆盖率测试监视和报告各种 i/o 请求数据包 (Irp) 为指定设备进入或离开驱动程序堆栈。 覆盖率测试中的数据可帮助确定驱动程序测试和验证过程中的覆盖率弱点。
