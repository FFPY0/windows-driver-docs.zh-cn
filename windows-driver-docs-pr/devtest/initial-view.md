打开 "TraceView" 窗口时，" [跟踪会话" 列表](trace-session-list.md) 将显示，但该列表为空。 以下屏幕截图显示了一个空的 TraceView 窗口。

![空 traceview 窗口的屏幕截图](images/traceview-empty.png)
