# <a name="pnpcpu-general-notes"></a>PNPCPU 常规说明


PNPCPU 目录应包含：

-   总线.遵从

-   BUSENUM.SYS

-   DELTA.CAT

-   PNPCPU.EXE

-   PNPCPU.SYS

-   双核处理器.遵从

运行 PNPCPU 工具不需要特定路径设置，尽管需要从包含以上列出的所有文件的目录运行 Pnpcpu.exe。
