# <a name="static-driver-verifier-options-file"></a>静态驱动程序验证程序选项文件


静态驱动程序验证程序选项文件（sdv-default.xml）存储静态驱动程序验证程序的默认选项设置。 此文件是在 WDK 的 \\ tools \\ sdv \\ data \\ wdm 和 \\ tools \\ sdv \\ data \\ wdf 子目录中创建的。 你可以编辑该文件以更改选项设置，并将其复制到任何驱动程序的源目录。
