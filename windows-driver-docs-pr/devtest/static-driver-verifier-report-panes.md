# <a name="static-driver-verifier-report-panes"></a>静态驱动程序验证程序报告窗格


静态驱动程序验证程序报表包含以下窗格：

-   [跟踪树窗格](trace-tree-pane.md)

-   [源代码窗格](source-code-pane.md)

-   [状态窗格](state-pane.md)

" **跟踪树** " 窗格、" **源代码** " 窗格和 " **状态** " 窗格一起构成了 " [缺陷查看器](defect-viewer.md) " 窗口。

仅当 SDV 检测到 "缺陷" (规则冲突) 时，才可以打开缺陷查看器。
