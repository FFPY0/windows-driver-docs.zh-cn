# <a name="static-driver-verifier-rule"></a>静态驱动程序验证程序规则


验证规则定义在操作系统的驱动程序和内核接口之间进行适当交互的要求。 规则以规范语言编写，用于 (SLIC) 的界面检查，这是 Microsoft Research 专门为此目的开发的一种简单语言。
