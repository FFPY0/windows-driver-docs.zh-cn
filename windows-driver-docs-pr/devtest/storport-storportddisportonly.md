# <a name="storportddisportonly-rule-storport"></a>StorPortDDIsPortOnly 规则 (storport) 


此规则包含 StorPort 仅限 DDIs 的 (的列表，其中包含不应在 StorPort 微型端口中调用的互锁函数) 。

**驱动程序模型： Storport**
