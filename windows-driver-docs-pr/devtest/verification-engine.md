# <a name="verification-engine"></a>验证引擎


SDV *验证引擎* 是一种功能强大的证明引擎，可对 SDV 执行规则验证。

验证引擎采用以下输入：

-   驱动程序代码

-   [静态驱动程序验证程序规则](static-driver-verifier-rule.md)

-   静态驱动程序验证程序 [操作系统模型](operating-system-model.md)
