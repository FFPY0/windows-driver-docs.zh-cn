# <a name="wdf-ddi-fault-injection-tester-wdffitester"></a>WDF DDI 错误注入测试程序 (WdfFiTester)


WdfFiTester 使用 WMI 接口来配置错误注入到 KMDF 设备驱动程序接口中 (DDI) 函数调用。 WdfFiTester 测试 KMDF 驱动程序的功能，以正确处理这些注入的故障。
