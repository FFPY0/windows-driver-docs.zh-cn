# <a name="checkdeviceobjectflags-rule-wdm"></a>CheckDeviceObjectFlags 规则 (wdm) 


**CheckDeviceObjectFlags** 规则指定总线驱动程序必须检查用于 do power PAGABLE 的设备对象标志 \_ \_ ，并 \_ \_ 为 FDO 和子 PDOs 一致地设置电源浪涌。 此规则仅适用于总线驱动程序。

**驱动程序模型： WDM**
