# <a name="irpprocessingcomplete-rule-wdm"></a>IrpProcessingComplete 规则 (wdm) 


**IrpProcessingComplete** 规则指定如果派单例程返回状态 \_ SUCCESS，则 IRP 必须已由驱动程序本身或较低级别的驱动程序完成。

**驱动程序模型： WDM**
