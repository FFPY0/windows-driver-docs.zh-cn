# <a name="irqlrtlpassive-rule-wdm"></a>IrqlRtlPassive 规则 (wdm) 


IrqlRtlPassive 规则指定仅当驱动程序在 IRQL = 被动级别执行时才调用 [**RtlDeleteRegistryValue**](/windows-hardware/drivers/ddi/wdm/nf-wdm-rtldeleteregistryvalue) \_ 。

**驱动程序模型： WDM**

**Bug 检查 () 发现此规则**： [**bug 检查0XC4：驱动程序 \_ 验证器 \_ 检测到 \_ 违反**](../debugger/bug-check-0xc4--driver-verifier-detected-violation.md) (0x0002001E)
