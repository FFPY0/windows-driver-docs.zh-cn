# <a name="dcibeginaccess"></a>DCIBeginAccess


## <span id="ddk_dcibeginaccess_gg"></span><span id="DDK_DCIBEGINACCESS_GG"></span>


请勿使用此函数;请改用最新的 DirectX 图形界面与 DirectDraw 支持的驱动程序通信。
