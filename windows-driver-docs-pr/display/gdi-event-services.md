# <a name="gdi-event-services"></a>GDI 事件服务


## <span id="ddk_gdi_event_services_gg"></span><span id="DDK_GDI_EVENT_SERVICES_GG"></span>


GDI 提供了若干与事件相关的服务。 使用这些服务的驱动程序可以创建和删除事件、映射和取消映射事件，以及读取、设置和清除事件。
