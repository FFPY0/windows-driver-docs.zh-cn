# <a name="optional-display-driver-functions"></a>可选的显示驱动程序函数


## <span id="ddk_optional_display_driver_functions_gg"></span><span id="DDK_OPTIONAL_DISPLAY_DRIVER_FUNCTIONS_GG"></span>

位图管理函数

绘制函数

图像颜色管理功能

指针和窗口管理函数

杂项函数
