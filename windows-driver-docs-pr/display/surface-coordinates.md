# <a name="surface-coordinates"></a>图面坐标


## <span id="ddk_surface_coordinates_gg"></span><span id="DDK_SURFACE_COORDINATES_GG"></span>


设备图面是2²⁸ x 2 ²⁸像素的数组的子集。 这些像素由28位有符号数字对，为设备图面的左上角像素指定坐标 (0，0) 。 设备图面位于此坐标空间的右下象限中，这两个坐标都是非负数。
