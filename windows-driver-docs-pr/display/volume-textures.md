# <a name="volume-textures"></a>体积纹理


## <span id="ddk_volume_textures_gg"></span><span id="DDK_VOLUME_TEXTURES_GG"></span>


DirectX 8.0 添加了对卷或3D 纹理的支持。 此类纹理除了具有宽度和高度之外，还具有深度。
