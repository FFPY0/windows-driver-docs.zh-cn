# <a name="diagnostics-and-debugging"></a>诊断和调试


## <span id="ddk_diagnostics_and_debugging_if"></span><span id="DDK_DIAGNOSTICS_AND_DEBUGGING_IF"></span>


RDBSS 提供了许多用于诊断和调试的例程。 这些例程分为两类：

-   断言和调试例程

-   引用和取消引用跟踪例程

这些例程包括下表中的项。
