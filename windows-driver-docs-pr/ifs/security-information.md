# <a name="security_information"></a>安全 \_ 信息


安全 \_ 信息

``` syntax
typedef ULONG SECURITY_INFORMATION, *PSECURITY_INFORMATION;
```




类型为 \_ "安全信息" 的值用于标识正在设置或查询的与对象相关的安全信息。 此安全信息包括：

-   对象的所有者

-   对象的主要组

-   对象的自由访问控制列表 (DACL) 

-   对象的系统 ACL (SACL) 

安全信息的每一项都由一个位标志来指定。 以下值指定位。
