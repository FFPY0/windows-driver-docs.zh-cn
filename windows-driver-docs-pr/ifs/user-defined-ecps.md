# <a name="user-defined-ecps"></a>用户定义的 ECP


若要定义自己的 ECP，请定义一个 GUID 来标识 ECP，并定义描述 ECP 的上下文结构。 有关如何在驱动程序代码中定义 Guid 和使用它们的信息，请参阅在 [驱动程序中使用 guid](../kernel/using-guids-in-drivers.md)。
