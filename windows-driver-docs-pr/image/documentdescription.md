# <a name="documentdescription-element"></a>DocumentDescription 元素


必需的 **DocumentDescription** 元素定义所有与当前标识的 [**文档**](document.md) 元素的基本创建信息相关的说明属性。

<a name="usage"></a>使用情况
-----

```xml
<wscn:DocumentDescription>
  child elements
</wscn:DocumentDescription>
```

<a name="attributes"></a>特性
----------

没有特性。
