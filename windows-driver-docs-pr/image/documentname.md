# <a name="documentname-element"></a>System.drawing.printing.printdocument.documentname 元素


必需的 **system.drawing.printing.printdocument.documentname** 元素包含客户端提供的文档的名称。

<a name="usage"></a>使用情况
-----

```xml
<wscn:DocumentName>
  text
</wscn:DocumentName>
```

<a name="attributes"></a>特性
----------

没有特性。

<a name="text-value"></a>文本值
----------

必需。 任何字符串。
