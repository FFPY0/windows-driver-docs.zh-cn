# <a name="object-size-in-ptp"></a>PTP 中的对象大小





ObjectInfo 数据集的 ObjectCompressedSize 字段在 PIMA INDIAN DIABETES 15740 标准中是可选的，但 Microsoft PTP WIA 微型驱动程序要求此字段为非零值。
