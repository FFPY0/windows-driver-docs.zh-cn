# <a name="required-root-item-properties-for-scanner-storage"></a>扫描仪存储的必需根项属性


所有 WIA 扫描器项树的根项都需要支持相同的属性，无论扫描程序是否支持存储。 有关必需的根项属性的详细信息，请参阅 [WIA 扫描器的必需的根项属性](required-root-item-properties-for-wia-scanners.md)。
