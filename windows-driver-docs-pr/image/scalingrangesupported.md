# <a name="scalingrangesupported-element"></a>ScalingRangeSupported 元素


必需的 **ScalingRangeSupported** 元素描述扫描设备支持的、用于缩放输出文档的值的范围。

<a name="usage"></a>使用情况
-----

```xml
<wscn:ScalingRangeSupported>
  child elements
</wscn:ScalingRangeSupported>
```

<a name="attributes"></a>特性
----------

没有特性。
