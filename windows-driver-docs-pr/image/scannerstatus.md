# <a name="scannerstatus-element"></a>ScannerStatus 元素


必需的 **ScannerStatus** 元素包含与扫描程序相关的信息的当前状态，这些信息自动机时出错 (如扫描器) 控件中的时间和更改条件。

<a name="usage"></a>使用情况
-----

```xml
<wscn:ScannerStatus>
  child elements
</wscn:ScannerStatus>
```

<a name="attributes"></a>特性
----------

没有特性。
