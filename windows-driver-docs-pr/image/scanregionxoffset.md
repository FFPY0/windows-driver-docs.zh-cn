# <a name="scanregionxoffset-element"></a>ScanRegionXOffset 元素


可选的 **ScanRegionXOffset** 元素指定从快速扫描导引边缘到扫描区域开头之间的距离。

<a name="usage"></a>使用情况
-----

```xml
<wscn:ScanRegionXOffset wscn:MustHonor=""                        wscn:Override=""                        wscn:UsedDefault=""
  MustHonor = "xs:string"
  Override = "xs:string"
  UsedDefault = "xs:string">
  text
</wscn:ScanRegionXOffset wscn:MustHonor=""                        wscn:Override=""                        wscn:UsedDefault="">
```

<a name="remarks"></a>备注
-------

有关扫描区域参数的详细信息，请参阅 [**ScanRegion**](scanregion.md)。

仅当 **ScanRegionXOffset** 元素包含在 **CreateScanJobRequest** 层次结构内时，客户端才能指定可选的 **MustHonor** 属性。 有关 **MustHonor** 及其用法的详细信息，请参阅 [**CreateScanJobRequest**](createscanjobrequest.md)。

仅当 **ScanRegionXOffset** 元素包含在 **DocumentFinalParameters** 层次结构中时，WSD 扫描服务才能指定可选 **Override** 和 **UsedDefault** 属性。 有关 **Override** 和 **UsedDefault** 及其用法的详细信息，请参阅 [**DocumentFinalParameters**](documentfinalparameters.md)。
