# <a name="twain-compatibility"></a>TWAIN 兼容性





WIA 微型驱动程序可以支持其他图像处理体系结构，如 TWAIN。 从 Windows Me 开始，Microsoft 不再随 Windows 一起提供 TWAIN 数据源。 对于扫描仪和数字照相机内置驱动程序，Microsoft 仅接受 WIA 驱动程序。
