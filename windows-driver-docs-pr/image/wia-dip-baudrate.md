# <a name="wia_dip_baudrate"></a>WIA \_ DIP \_ 波特率


WIA \_ DIP \_ 波特率属性包含设备的当前波特率设置。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_baudrate_si"></span><span id="DDK_WIA_DIP_BAUDRATE_SI"></span>


属性类型： VT \_ BSTR

有效值： WIA " \_ \_ 无"

访问权限：只读

<a name="remarks"></a>备注
-------

\_ \_ 如果设备未通过串行电缆连接，则 WIA DIP 波特率属性的值应为 "空"。
