# <a name="wia_dip_server_name"></a>WIA \_ DIP \_ 服务器 \_ 名称


WIA \_ DIP \_ 服务器 \_ 名称属性包含 wia 微型驱动程序正在其上运行的服务器的名称。

## <span id="ddk_wia_dip_server_name_si"></span><span id="DDK_WIA_DIP_SERVER_NAME_SI"></span>


属性类型： VT \_ BSTR

有效值： WIA " \_ \_ 无"

访问权限：只读

<a name="remarks"></a>备注
-------

WIA \_ DIP 服务器名称的默认 \_ 值 \_ 为 "本地"。 当应用程序连接到同一台计算机上的设备时，此属性应包含字符串 "local"。
