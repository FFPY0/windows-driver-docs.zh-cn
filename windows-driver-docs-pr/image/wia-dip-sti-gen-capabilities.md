# <a name="wia_dip_sti_gen_capabilities"></a>WIA \_ DIP \_ STI \_ GEN \_ 功能


WIA \_ DIP \_ STI \_ GEN \_ 功能属性包含设备的通用 STI 功能，这些功能是从驱动程序的 INF 文件中获取的。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_sti_gen_capabilities_si"></span><span id="DDK_WIA_DIP_STI_GEN_CAPABILITIES_SI"></span>


属性类型： VT \_ I4

有效值： WIA " \_ \_ 无"

访问权限：只读

<a name="remarks"></a>备注
-------

应用程序读取 WIA \_ DIP \_ STI \_ GEN \_ 功能属性来确定设备的一般 STI 功能。
