# <a name="wia_dip_wia_version"></a>WIA \_ DIP \_ WIA \_ 版本


"WIA \_ DIP \_ wia \_ 版本" 属性包含的数字 (为计算机上安装的当前 WIA 版本的字符串) 。 WIA 服务创建并维护此属性。

## <span id="ddk_wia_dip_wia_version_si"></span><span id="DDK_WIA_DIP_WIA_VERSION_SI"></span>


属性类型： VT \_ BSTR

有效值： WIA " \_ \_ 无"

访问权限：只读

<a name="remarks"></a>备注
-------

应用程序将读取 WIA \_ DIP \_ wia \_ 版本，以确定计算机上安装的 WIA 版本。
