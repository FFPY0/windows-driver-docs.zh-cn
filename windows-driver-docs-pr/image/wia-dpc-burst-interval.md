# <a name="wia_dpc_burst_interval"></a>WIA \_ DPC \_ 突发 \_ 间隔


"WIA \_ DPC \_ 突发 \_ 间隔" 属性包含突发操作期间映像捕获之间的时间（以毫秒为单位）。

## <span id="ddk_wia_dpc_burst_interval_si"></span><span id="DDK_WIA_DPC_BURST_INTERVAL_SI"></span>


属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ NONE、wia 内容 \_ \_ 列表或 wia 的内容 \_ \_ 范围

访问权限：读/写
