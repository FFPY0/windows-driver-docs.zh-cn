# <a name="wia_dpc_contrast"></a>WIA \_ DPC \_ 对比度


WIA \_ DPC \_ 对比度属性指示捕获的映像的感知对比度。

## <span id="ddk_wia_dpc_contrast_si"></span><span id="DDK_WIA_DPC_CONTRAST_SI"></span>


属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表或 wia 的 \_ 内容 \_ 范围

访问权限：读/写

<a name="remarks"></a>备注
-------

WIA \_ DPC \_ 对比度属性可以包含值的列表或值的范围。

WIA DPC 对比度支持的最小值 \_ \_ 表示对比度的最小值，最大值表示最高对比度。 通常，范围中间的值表示正常或默认值对比度。
