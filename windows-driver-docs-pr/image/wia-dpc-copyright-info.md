# <a name="wia_dpc_copyright_info"></a>WIA \_ DPC \_ 版权 \_ 信息


WIA \_ DPC \_ 版权 \_ 信息属性包含版权通知。

## <span id="ddk_wia_dpc_copyright_info_si"></span><span id="DDK_WIA_DPC_COPYRIGHT_INFO_SI"></span>


属性类型： VT \_ BSTR

有效值： WIA " \_ \_ 无"

访问权限：读/写

<a name="remarks"></a>备注
-------

设备使用 "WIA \_ DPC \_ 版权信息" \_ 属性在捕获的每个 EXIF 映像中填充版权字段。
