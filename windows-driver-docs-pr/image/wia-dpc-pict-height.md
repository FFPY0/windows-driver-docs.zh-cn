# <a name="wia_dpc_pict_height"></a>WIA \_ DPC \_ PICT \_ 高度


"WIA \_ DPC \_ PICT \_ 高度" 属性包含用于新捕获图像的高度（以像素为单位）。

## <span id="ddk_wia_dpc_pict_height_si"></span><span id="DDK_WIA_DPC_PICT_HEIGHT_SI"></span>


属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表或 wia 的 \_ 内容 \_ 范围

访问权限：读/写

<a name="remarks"></a>备注
-------

WIA dpc "pict 高度" 属性的有效值列表 \_ \_ \_ 与 " [**wia \_ dpc \_ pict \_ 宽度**](wia-dpc-pict-width.md) " 属性的有效值列表一一对应。 如果单独的宽度和高度以线性方式进行设置，则可以将其表示为范围。
