# <a name="wia_dpc_pictures_taken"></a>已 \_ 拍摄 WIA DPC \_ 图片 \_


"WIA \_ DPC \_ 图片 \_ 拍摄量" 属性包含照相机拍摄的图片数。 WIA 微型驱动程序创建并维护此属性。

属性类型： VT \_ I4

有效值： WIA " \_ \_ 无"

访问权限：只读
