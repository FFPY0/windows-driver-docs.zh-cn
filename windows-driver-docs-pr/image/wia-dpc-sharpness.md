# <a name="wia_dpc_sharpness"></a>WIA \_ DPC \_ 清晰度


WIA \_ DPC \_ 清晰度属性指示捕获的映像的已感知清晰度。

## <span id="ddk_wia_dpc_sharpness_si"></span><span id="DDK_WIA_DPC_SHARPNESS_SI"></span>


属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表或 wia 的 \_ 内容 \_ 范围

访问权限：读/写

<a name="remarks"></a>备注
-------

WIA \_ DPC \_ 清晰度属性可以使用值的列表或值的范围。 最小值表示锐度的最小值，最大值表示最大清晰度。 通常，范围中间的值表示正常或默认的清晰度。
