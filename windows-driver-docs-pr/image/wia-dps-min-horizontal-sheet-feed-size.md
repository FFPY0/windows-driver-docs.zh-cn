# <a name="wia_dps_min_horizontal_sheet_feed_size"></a>WIA \_ DPS \_ 最小 \_ 水平 \_ 页面 \_ 源 \_ 大小


WIA \_ DPS \_ MIN \_ 横坐标 \_ \_ \_ page SIZE 属性包含扫描仪的文档送纸器可以扫描的最小页面的物理水平尺寸，以英寸 ( 001) 。 WIA 微型驱动程序创建并维护此属性。
