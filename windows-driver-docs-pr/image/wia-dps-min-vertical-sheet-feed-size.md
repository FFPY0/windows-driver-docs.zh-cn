# <a name="wia_dps_min_vertical_sheet_feed_size"></a>WIA \_ DPS \_ 最小 \_ 垂直 \_ 纸张 \_ 馈送 \_ 大小


"WIA \_ DPS \_ 最 \_ 小 \_ 垂直 \_ 进纸 \_ 大小" 属性包含扫描程序的文档送纸器可以扫描的最小页面的物理垂直尺寸，以英寸 ( 001) 为单位。 WIA 微型驱动程序创建并维护此属性。

## <span id="ddk_wia_dps_min_vertical_sheet_feed_size_si"></span><span id="DDK_WIA_DPS_MIN_VERTICAL_SHEET_FEED_SIZE_SI"></span>


属性类型： VT \_ I4

有效值： WIA " \_ \_ 无"

访问权限：只读
