# <a name="wia_dps_preview"></a>WIA \_ DPS \_ 预览版


WIA \_ DPS \_ PREVIEW 属性指示设备的预览模式。 应用程序将此属性设置为将设备置于预览模式。

## <span id="ddk_wia_dps_preview_si"></span><span id="DDK_WIA_DPS_PREVIEW_SI"></span>


属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表

访问权限：读/写
