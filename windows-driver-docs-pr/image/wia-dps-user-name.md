# <a name="wia_dps_user_name"></a>WIA \_ DPS \_ 用户名 \_


使用 \_ \_ "wia 驱动程序根" 项的 "wia DPS 用户名" \_ 属性，驱动程序可以获取域名，或在给定时间点访问 WIA 设备的用户的计算机名称和用户名。 WIA 服务创建并维护此属性。

属性类型： VT \_ BSTR

有效值： WIA " \_ \_ 无"

访问权限：只读
