# <a name="wia_ips_auto_deskew"></a>WIA \_ IPS \_ 自动 \_ 反扭曲


WIA \_ IPS \_ 自动 \_ 反扭曲属性指示设备是否应使用自动倾斜校正。 WIA 微型驱动程序创建并维护此属性。

属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表

访问权限：读/写

<a name="remarks"></a>备注
-------

下表描述了对 WIA \_ IPS \_ 自动 \_ 反扭曲属性有效的常量。

<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th>值</th>
<th>定义</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>WIA_AUTO_DESKEW_ON</p></td>
<td><p>使用自动倾斜校正。</p></td>
</tr>
<tr class="even">
<td><p>WIA_AUTO_DESKEW_OFF</p></td>

</tr>
