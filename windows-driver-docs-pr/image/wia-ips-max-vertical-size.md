# <a name="wia_ips_max_vertical_size"></a>WIA \_ IPS \_ 最大 \_ 垂直 \_ 大小


"WIA \_ IPS \_ 最大 \_ 垂直 \_ 大小" 属性包含扫描仪的平板的物理垂直尺寸，以英寸 ( 001) 为单位。 WIA 微型驱动程序创建并维护此属性。

属性类型： VT \_ I4

有效值： WIA " \_ \_ 无"

访问权限：只读
