# <a name="wia_ips_maximum_barcode_search_retries"></a>WIA \_ IPS \_ 最大 \_ 条形码 \_ 搜索 \_ 重试次数


**WIA \_ IPS \_ 最大 \_ 条形码 \_ 搜索 \_ 重试** 属性描述当启用条形码检测时找不到条形码时，读取器尝试的最大重试次数。



属性类型： VT \_ UI4

有效值： WIA 内容 \_ \_ 范围

访问权限：读/写

<a name="remarks"></a>备注
-------

所有条形码读取器项都需要此属性。 可以实现属性以支持包含一个单个值的范围，包括 0 (不) 重试。
