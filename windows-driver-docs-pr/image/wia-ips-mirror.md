# <a name="wia_ips_mirror"></a>WIA \_ IP \_ 镜像


\_ \_ MICROSOFT 保留 WIA ip 镜像属性以供将来使用，目前未实现。

## <span id="ddk_wia_ips_mirror_si"></span><span id="DDK_WIA_IPS_MIRROR_SI"></span>


属性类型： VT \_ I4

有效值： WIA " \_ \_ 无"

访问权限：只读

<a name="remarks"></a>备注
-------

WIA \_ ip \_ 镜像属性与 [**wia \_ ip \_ 定位**](wia-ips-orientation.md) 属性无关。 WIA \_ ip \_ 方向指定要在与扫描方向之间进行扫描的文档方向; 相反，wia \_ ip \_ 镜像指定在扫描映像后要对其应用的操作。
