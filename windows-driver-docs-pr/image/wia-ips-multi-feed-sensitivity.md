# <a name="wia_ips_multi_feed_sensitivity"></a>WIA \_ IPS \_ 多 \_ 源 \_ 敏感度


" **WIA \_ IPS \_ 多 \_ 源 \_ 敏感度** " 属性用于将多源检测触发器更改为设备支持的最低和最高敏感度之间较低或较高的值。 WIA 微型驱动程序创建并维护此属性。




属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 范围

访问权限：读/写

<a name="remarks"></a>备注
-------

此属性是可选的，并且仅对作为 WIA 类别送纸器 (显示在 " [**wia \_ IPA \_ 项 \_ 类别**](wia-ipa-item-category.md) " 属性中的 "输送器" 数据源项目中有效的 "wia \_ 分类 \_ 送纸器") [**\_ \_ \_**](wia-ips-multi-feed.md) \_ \_ \_ \_ 。
