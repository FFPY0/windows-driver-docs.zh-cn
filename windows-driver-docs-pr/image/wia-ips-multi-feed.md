# <a name="wia_ips_multi_feed"></a>WIA \_ IPS \_ 多 \_ 源


**Wia \_ IPS \_ 多 \_ 源** 属性用于配置当在设备上检测到多个源条件时，wia 微型驱动程序要执行的操作。 WIA 微型驱动程序创建并维护此属性。




属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表

访问权限：读/写
