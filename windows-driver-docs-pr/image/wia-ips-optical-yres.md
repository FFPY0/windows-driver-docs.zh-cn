# <a name="wia_ips_optical_yres"></a>WIA \_ IP \_ 光纤 \_ YRES


WIA \_ IPS \_ \_ YRES 属性包含设备支持的最高垂直光纤分辨率（以每英寸点数为单位） (dpi) 。 WIA 微型驱动程序创建并维护此属性。

属性类型： VT \_ I4

有效值： WIA " \_ \_ 无"

访问权限：只读
