# <a name="wia_ips_page_size"></a>WIA \_ IP \_ 页面 \_ 大小


"WIA \_ ip \_ 页面 \_ 大小" 属性包含当前选定要扫描的页面的大小。

属性类型： VT \_ I4

有效值： WIA 内容 \_ \_ 列表

访问权限：读/写

<a name="remarks"></a>备注
-------

若要选择要扫描的页的维度，应用程序将设置 "WIA \_ ip \_ 页面大小" \_ 属性。 WIA 微型驱动程序创建并维护此属性。
