# <a name="wia_ips_printer_endorser_num_lines"></a>WIA \_ IPS \_ 打印机 \_ ENDORSER \_ NUM \_ LINES


**WIA \_ IPS \_ PRINTER \_ ENDORSER \_ NUM \_ LINES** 属性指定了 Imprinter/ENDORSER 单位可以在文档的每一侧打印或认可的最大行数。 此属性由 WIA 迷你驱动程序初始化和维护，并且可用于 Windows 8 及更高版本的 Windows。

属性类型： VT \_ UI4

有效值： WIA " \_ \_ 无"

访问权限：只读

<a name="remarks"></a>备注
-------

对于 Imprinter/Endorser 项， **WIA \_ IPS \_ PRINTER \_ ENDORSER \_ NUM \_ LINES** 属性是可选的。 如果此属性不受支持，则只允许在文档的每一侧使用一行文本。
