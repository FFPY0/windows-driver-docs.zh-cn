# <a name="wsd-scan-service-operation-elements"></a>WSD 扫描服务操作元素

不能扩展 WSDL 服务说明中的操作列表。 如果要实现非标准操作，则这些操作必须是为 WSD 扫描服务定义扩展的新 **portType** 类型的一部分。
