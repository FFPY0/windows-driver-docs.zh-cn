-   与 Setupapi.log 中定义的 SPCRP_ *Xxx* 标识符对应的 [设备安装程序类属性](/previous-versions/ff542239(v=vs.85)) *。*

-   没有 SPCRP_ *Xxx* 标识符，但具有相应系统定义的注册表项值的属性。
