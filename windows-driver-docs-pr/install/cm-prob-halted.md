# <a name="code-44---cm_prob_halted"></a>代码 44 - CM_PROB_HALTED

此设备管理器错误消息表示设备已停止。

## <a name="error-code"></a>错误代码

44

### <a name="display-message"></a>显示消息

"应用程序或服务已关闭此硬件设备。  (代码 44) "

### <a name="recommended-resolution"></a>推荐的解决方案

您必须重新启动计算机。
