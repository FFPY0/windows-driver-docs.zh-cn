<a name="remarks"></a>备注
-------

Windows 将 DEVPKEY_Device_RemovalPolicy 的值设置为 Cfgmgr32 中定义的 CM_REMOVAL_POLICY_ *Xxx* 值之一。

可以调用 [**SetupDiGetDeviceProperty**](/windows/win32/api/setupapi/nf-setupapi-setupdigetdevicepropertyw) 来检索 DEVPKEY_Device_RemovalPolicy 属性的值。

Windows Server 2003 和 Windows XP 支持此属性，但不支持 DEVPKEY_Device_RemovalPolicy 属性键。 相反，你可以使用相应的 SPDRP_REMOVAL_POLICY 标识符来访问这些早期版本的 Windows 上的属性值。 有关如何在这些早期版本的 Windows 上访问此属性值的信息，请参阅 [SPDRP_Xxx 属性访问设备实例](./accessing-device-instance-spdrp-xxx-properties.md)。
