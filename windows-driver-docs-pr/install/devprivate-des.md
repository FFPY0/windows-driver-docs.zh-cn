# <a name="devprivate_des"></a>DEVPRIVATE_DES


**注意**  此结构仅供内部使用。

 

DEVPRIVATE_DES 结构用于指定资源列表或资源需求列表，该列表描述设备实例的专用于设备的专用资源使用情况。 有关资源列表和资源要求列表的详细信息，请参阅 [硬件资源](../kernel/hardware-resources.md)。
