此 DIF 代码已过时，并且在 Microsoft Windows 2000 和更高版本的 Windows 中不再受支持。

Windows 只使用 DIF_NEWDEVICEWIZARD_ *XXX* 请求，如 [**DIF_NEWDEVICEWIZARD_FINISHINSTALL**](dif-newdevicewizard-finishinstall.md)。
