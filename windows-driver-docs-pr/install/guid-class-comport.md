# <a name="guid_class_comport"></a>GUID_CLASS_COMPORT


对于支持 16550 UART 兼容硬件接口的设备，GUID_CLASS_COMPORT 是设备接口类的过时标识符。 使用此类的新实例 [**GUID_DEVINTERFACE_COMPORT**](guid-devinterface-comport.md) 类标识符。
