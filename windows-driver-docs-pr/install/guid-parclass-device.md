# <a name="guid_parclass_device"></a>GUID_PARCLASS_DEVICE


GUID_PARCLASS_DEVICE 是附加到并行端口的设备的设备接口类的过时标识符。 对于此类的新实例，请改用 [**GUID_DEVINTERFACE_PARCLASS**](guid-devinterface-parclass.md) 类标识符。
