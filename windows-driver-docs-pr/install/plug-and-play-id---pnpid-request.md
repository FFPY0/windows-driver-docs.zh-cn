# <a name="plug-and-play-id---pnpid-request"></a>即插即用 ID - PNPID 请求


即插即用 ID 现在通过 UEFI.org 网站请求。 若要请求 ID，请参阅 [PNP id AND ACPI Id Registry](https://uefi.org/PNP_ACPI_Registry)。
