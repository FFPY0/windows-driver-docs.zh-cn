# <a name="sample-device-installation-files"></a>设备安装文件示例





Windows 驱动程序工具包 (WDK) 包含各种示例安装文件，如 INF 文件、设备安装应用程序、类安装程序、共同安装程序和属性页提供程序。

大多数示例安装文件位于 WDK 的 " *src \\ 安装程序* " 子目录下。 另请参阅 WDK 的 *src \\ general \\ toaster* 子目录。
