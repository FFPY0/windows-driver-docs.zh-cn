# <a name="sp_installwizard_data"></a>SP_INSTALLWIZARD_DATA


此结构已过时。

SP_INSTALLWIZARD_DATA 是 **SetupDiGetWizardPage** 函数的参数。

Windows 使用 DIF_NEWDEVICEWIZARD_ *XXX* 请求，如 DIF_NEWDEVICEWIZARD_FINISHINSTALL，而不是 DIF_INSTALLWIZARD。
