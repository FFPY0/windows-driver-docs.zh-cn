# <a name="starting-a-device-in-a-filter-driver"></a>在筛选器驱动程序中启动设备





上层筛选器驱动程序可能会增加函数驱动程序的任何启动活动。

较低级别的筛选器通常会补充设备的功能，并且可能会参与设备的启动。
