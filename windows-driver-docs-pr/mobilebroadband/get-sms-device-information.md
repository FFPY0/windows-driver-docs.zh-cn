# <a name="get-sms-device-information"></a>获取短信设备信息


移动宽带 SMS 平台提供有关移动宽带设备的信息，包括帐户电话号码、状态和蜂窝电话号码。

**用于获取 SMS 设备移动电话号码的 JavaScript 代码示例**

``` syntax
var mobileNumber = smsDevice.accountPhoneNumber;
```
