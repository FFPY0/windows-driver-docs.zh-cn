# <a name="live-tile-updates"></a>动态磁贴更新


在 Windows 10 " **开始** " 菜单上，磁贴是应用的主要表示形式。 用户通过这些磁贴启动其应用，磁贴可以向用户显示新的相关内容和定制的内容。 可以使磁贴保持活动状态，并将用户重复移回到应用。 您可以显示有关 [高级服务](premium-services.md)的信息，或者您想要让用户注意的重要信息。
