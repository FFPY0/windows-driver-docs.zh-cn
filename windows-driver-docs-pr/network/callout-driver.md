# <a name="callout-driver"></a>标注驱动程序


*标注驱动程序* 是一个内核模式驱动程序，可实现一个或多个 [标注](callout.md)。 标注驱动程序使用 [筛选器引擎](filter-engine.md) 注册其标注，以便当计算机处理连接或数据包时，筛选器引擎可以调用标注的 [标注函数](callout-function.md) 。
