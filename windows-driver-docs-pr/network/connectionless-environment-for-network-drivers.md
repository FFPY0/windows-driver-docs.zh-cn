# <a name="connectionless-environment-for-network-drivers"></a>网络驱动程序的无连接环境





无连接环境是用于无连接媒体（如以太网）的标准网络驱动程序环境。 有关此环境中驱动程序的说明，请参阅 [Ndis 驱动](ndis-drivers.md) 程序和 [Ndis 小型端口驱动程序](ndis-miniport-drivers2.md)。
