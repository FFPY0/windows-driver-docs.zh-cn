# <a name="ndistapi-overview"></a>NDISTAPI 概述





NDISTAPI 是系统提供的驱动程序，用于将 NDISWAN 和 NDIS WAN 微型端口驱动程序接口到 TAPI 服务。 对于 NDIS 5.0 和更高版本的驱动程序，不支持 NDIS WAN 微型端口驱动程序。 新驱动程序应基于 CoNDIS WAN 驱动程序体系结构。
