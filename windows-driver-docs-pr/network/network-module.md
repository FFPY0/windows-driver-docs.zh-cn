# <a name="network-module"></a>网络模块


*网络模块* 是一种软件模块，用于实现网络堆栈中的特定功能，如数据链接接口、传输协议或网络应用程序。 网络模块可以是 [提供程序模块](provider-module.md)和/或 [客户端模块](client-module.md)，具体取决于它在网络堆栈中的位置。
