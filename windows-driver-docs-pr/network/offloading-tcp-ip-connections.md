# <a name="offloading-tcpip-connections"></a>卸载 TCP/IP 连接





在 NDIS 6.0 和更高版本中支持卸载 TCP/IP 连接。

NDIS TCP/IP 连接卸载接口启用诸如 TCP 烟囱卸载之类的服务。

有关卸载 TCP/IP 连接的详细信息，请参阅 [完整 Tcp 卸载](full-tcp-offload.md)中的 Tcp 烟囱卸载概述。
