# <a name="oid_gen_friendly_name"></a>OID \_ 生成 \_ 友好 \_ 名称


作为查询，OID 生成 \_ \_ 友好 \_ 名称 oid 返回微型端口适配器的友好名称。

<a name="remarks"></a>备注
-------

OID 生成 \_ \_ 友好 \_ 名称 oid 返回微型端口适配器的友好名称。

友好名称旨在帮助用户快速准确地识别微型端口适配器，例如，"PCI 以太网适配器" 和 "虚拟专用网络适配器" 被视为友好名称。
