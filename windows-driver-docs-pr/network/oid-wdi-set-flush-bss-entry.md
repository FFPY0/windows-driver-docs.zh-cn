# <a name="oid_wdi_set_flush_bss_entry"></a>OID \_ WDI \_ 设置 \_ 刷新 \_ BSS \_ 条目


OID \_ WDI \_ SET \_ flush \_ \_ 会向设备发送刷新适配器所维护的 BSS 条目列表。 只能在工作站端口上发送此命令。

| 范围 | 设置序列化任务 | 正常执行时间 (秒)  |
|-------|--------------------------|---------------------------------|
| 端口  | 否                       | 1                               |

 

## <a name="set-property-parameters"></a>设置属性参数


无其他参数。 标头中的数据足够了。
## <a name="set-property-results"></a>设置属性结果


无其他数据。 标头中的数据足够了。
