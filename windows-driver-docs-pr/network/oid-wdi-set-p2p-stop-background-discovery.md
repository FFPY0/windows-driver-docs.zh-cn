# <a name="oid_wdi_set_p2p_stop_background_discovery"></a>OID \_ WDI \_ 设置 \_ P2P \_ 停止 \_ 后台 \_ 发现


OID \_ WDI \_ SET \_ P2P \_ STOP \_ 后台 \_ 发现指示适配器取消后台发现并停止正在进行的任何活动扫描。

| 范围 | 设置序列化任务 | 正常执行时间 (秒)  | 影响数据吞吐量/延迟 |
|-------|--------------------------|---------------------------------|---------------------------------|
| 端口  | 否                       | 1                               | 否                              |

 

## <a name="set-property-parameters"></a>设置属性参数


无其他参数。 标头中的数据足够了。
## <a name="set-property-results"></a>设置属性结果


无其他数据。 标头中的数据足够了。
