# <a name="remote_ndis_initialize_msg"></a>远程 \_ NDIS \_ 初始化 \_ 消息


此消息由主机发送到远程 NDIS 设备，以便初始化网络连接。 它是通过控制通道发送的，并且仅在设备未处于通过远程 NDIS 初始化的状态时发送。
