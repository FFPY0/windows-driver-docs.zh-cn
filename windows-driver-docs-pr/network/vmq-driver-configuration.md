# <a name="vmq-driver-configuration"></a>VMQ 驱动程序配置





本部分描述了 NDIS 提供的用于确定虚拟机队列 (网络适配器的 VMQ) 功能的功能。

小型端口驱动程序在网络适配器初始化期间向 NDIS 和过量驱动程序报告其 VMQ 功能。

标准 INF 文件条目可以在安装驱动程序时启用或禁用 VMQ 功能。
