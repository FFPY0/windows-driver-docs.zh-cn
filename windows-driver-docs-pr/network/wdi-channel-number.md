# <a name="wdi_channel_number"></a>WDI_CHANNEL_NUMBER

WDI_CHANNEL_NUMBER 数据类型是一个用于定义频道号的 UINT32 值。

```c++
typedef UINT32 WDI_CHANNEL_NUMBER;
```

## <a name="requirements"></a>要求

**支持的最低客户端**： Windows 10

**支持的最低服务器**： Windows server 2016

**标头**： Wditypes. hpp
