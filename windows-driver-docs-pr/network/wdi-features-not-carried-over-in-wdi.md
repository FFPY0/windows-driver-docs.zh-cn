# <a name="features-not-carried-over-in-wdi"></a>未转入 WDI 的功能


以下功能在以前的本机 WLAN 驱动程序模型中可用，但在 WDI 中不可用。

-   软 AP
-   IBSS
-   Netmon

以下软 AP 必需功能将移植到 Wi-Fi 直接。

-   添加了 802.11 b。
-   已将软 AP 的所有 tethering 要求移到 Wi-Fi 直接。
