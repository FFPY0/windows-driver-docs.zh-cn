# <a name="wdi_tlv_channel_number"></a>WDI \_ TLV \_ 通道 \_ 号


WDI \_ tlv \_ 通道 \_ 号是包含通道号的 tlv。

## <a name="tlv-type"></a>TLV 类型


0x121

## <a name="length"></a>长度


UINT8 的大小 (以字节为单位) 。

## <a name="values"></a>值


| 类型  | 描述         |
|-------|---------------------|
| UINT8 | 频道号。 |
