# <a name="wdi_tlv_datapath_capabilities"></a>WDI \_ TLV \_ 数据路径 \_ 功能


WDI \_ tlv \_ 数据路径 \_ 功能是包含数据路径功能的 tlv。

## <a name="tlv-type"></a>TLV 类型


0xB9

## <a name="length"></a>长度


Sum (所有包含的元素的大小) 。
