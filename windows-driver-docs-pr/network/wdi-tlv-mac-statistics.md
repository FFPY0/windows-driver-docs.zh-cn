# <a name="wdi_tlv_mac_statistics"></a>WDI \_ TLV \_ MAC \_ 统计信息


WDI \_ tlv \_ mac \_ 统计信息是一个 tlv，其中包含 [OID \_ WDI \_ 获取 \_ 统计](./oid-wdi-get-statistics.md)信息的每个对等 MAC 统计信息。

## <a name="tlv-type"></a>TLV 类型


0Xa6<

## <a name="length"></a>长度


Sum (所有包含的元素的大小) 。
