# <a name="wdi_tlv_operating_class"></a>WDI \_ TLV \_ 操作 \_ 类


WDI \_ tlv \_ 操作 \_ 类是一个 tlv，其中包含通道的频带。

## <a name="tlv-type"></a>TLV 类型


0xFA

## <a name="length"></a>长度


UINT8 的大小 (以字节为单位) 。

## <a name="values"></a>值


| 类型  | 描述                       |
|-------|-----------------------------------|
| UINT8 | 通道的频带。 |
