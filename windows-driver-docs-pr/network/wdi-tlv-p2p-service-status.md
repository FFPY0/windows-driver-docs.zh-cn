# <a name="wdi_tlv_p2p_service_status"></a>WDI \_ TLV \_ P2P \_ 服务 \_ 状态


WDI \_ tlv \_ P2P \_ 服务 \_ 状态是包含服务的服务状态的 tlv。

## <a name="tlv-type"></a>TLV 类型


0xFB

## <a name="length"></a>长度


UINT8 的大小 (以字节为单位) 。

## <a name="values"></a>值


| 类型  | 描述                      |
|-------|----------------------------------|
| UINT8 | 服务的服务状态。 |
