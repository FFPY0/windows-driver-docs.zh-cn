# <a name="wdi_tlv_phy_statistics"></a>WDI \_ TLV \_ PHY \_ 统计信息


WDI \_ tlv \_ PHY \_ 统计信息是一个 tlv，其中包含 [OID \_ WDI \_ 获取 \_ 统计](./oid-wdi-get-statistics.md)信息的每个 PHY 统计信息。

## <a name="tlv-type"></a>TLV 类型


0xA7

## <a name="length"></a>长度


Sum (所有包含的元素的大小) 。
