# <a name="wdi_tlv_receive_filter_field-0x65"></a>WDI \_ TLV \_ 接收 \_ 筛选器 \_ 字段 (0x65) 


WDI \_ tlv \_ 接收 \_ 筛选器 \_ 字段是一个 tlv，其中包含网络标头中的一个字段的接收筛选器测试条件。

## <a name="tlv-type"></a>TLV 类型


0x65

## <a name="length"></a>长度


Sum (所有包含的元素的大小) 。
