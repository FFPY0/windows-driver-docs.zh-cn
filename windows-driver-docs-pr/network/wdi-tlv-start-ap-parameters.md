# <a name="wdi_tlv_start_ap_parameters"></a>WDI \_ TLV \_ 启动 \_ AP \_ 参数


WDI \_ tlv \_ 启动 \_ ap \_ 参数是一个 TLV，其中包含 [OID \_ WDI \_ 任务 \_ 启动 \_ AP](./oid-wdi-task-start-ap.md)的参数。

## <a name="tlv-type"></a>TLV 类型


0xAB

## <a name="length"></a>长度


Sum (所有包含的元素的大小) 。
