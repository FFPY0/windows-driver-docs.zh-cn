本部分介绍有关 PCMCIA 内存卡驱动程序的设计和操作的以下特殊注意事项：

-   PCMCIA 内存卡的内存窗口中 Windows 2000 和更高版本的操作系统的限制

-   读取和写入 PCMCIA 内存卡上的属性内存
