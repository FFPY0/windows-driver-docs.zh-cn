# <a name="acpi-power-meter-interface"></a>ACPI 功率表接口


一种 WDM 驱动程序， (PMI) 从底层硬件平台访问 (PMB) 信息的电源计量接口。 驱动程序通过使用 ACPI PMI 对象来实现此功能。 该平台使用 ACPI 4.0 电源计量对象公开此信息。

本部分提供了 ACPI PMI 的高级概述。
