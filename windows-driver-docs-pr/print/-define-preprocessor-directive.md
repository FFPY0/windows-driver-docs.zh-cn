# <a name="define-preprocessor-directive"></a>\#定义预处理器指令

```GDL
#Define: symbol
```

\#Define 指令将 *符号* 值添加到预处理器符号字典。 *符号* 可以是任何值令牌，并且是必需的。 可以多次添加同一符号。
