# <a name="else-conditional-preprocessor-directive"></a>\#Else 条件预处理器指令


```GDL
#Else:
```

\#Else 指令定义上一节的末尾和新条件节的开头。 如果未保留构造中的上一节，则保留部分。 否则将被删除。 不使用任何符号。
