# <a name="boolean-values"></a>布尔值





布尔值定义为等于预定义的常量 **TRUE** 或 **FALSE**。 按照约定，接受布尔值的所有 [打印机属性](printer-attributes.md) 的符号名称以问号字符结尾，如以下示例所示：

```cpp
*RotateRasterData?: FALSE
```
