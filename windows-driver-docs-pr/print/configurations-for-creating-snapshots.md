# <a name="configurations-for-creating-snapshots"></a>用于创建快照的配置


配置只是一个参数列表和分配给每个参数的当前值。 GDL 分析器接口需要将配置置于 XML 中，但为了简化，下面的伪代码示例演示了使用参数/值对列表的配置的典型表示形式。

```cpp
Today: Sunday
Weather: Cloudy
```
