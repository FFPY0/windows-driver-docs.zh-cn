# <a name="deviceid"></a>DeviceID


**DeviceID** 命令返回 ID)  (ID 的设备标识符，格式为 **PKEY \_ PNPX \_ GlobalIdentity**。 此密钥返回根设备 (UUID) 的全局唯一 ID。 输出缓冲区包含以 NULL 结尾的字符串，该字符串表示 UUID。

有关函数发现密钥的详细信息，请参阅 Microsoft Windows SDK 文档。
