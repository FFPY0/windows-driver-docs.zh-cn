# <a name="enumerations-and-xsd-defined-data-types"></a>枚举和 XSD 定义的数据类型


使用 GDL 模板定义的枚举导致枚举数据类型定义显示在 GDL 分析器生成的 XSD 架构中。 使用 GDL 模板的 XSD 定义数据类型使 XSD 数据类型定义显示在 GDL 分析器生成的 XSD 架构中。
