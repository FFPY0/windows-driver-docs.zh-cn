# <a name="gdl-comments"></a>GDL 注释


*注释* 可以出现在 [非文本空白](gdl-whitespace-characters.md)可显示的任何位置。 注释以星号和百分号 (\* % ) 开头，并在遇到 linebreak 序列或继续 linebreak 时结束。

整个注释被视为非文本空白。 终止 linebreak 序列不是注释的一部分。 注释可以出现在嵌套的上下文中，但不会在注释内识别上下文。
