# <a name="gdl-directives-for-macros"></a>用于宏的 GDL 指令


GDL 包含用于定义宏的指令。 对宏的引用以行扩展，以获取由分析器解释的项。

GDL 包含以下宏指令：

-   **\* 宏** 定义一组值宏。

-   **\* BlockMacro** 定义了一个块宏。

-   **\* InsertBlock** 引用块宏

-   **=** (等号) 当后跟值宏的名称时，将引用值 sign。

有关宏指令的详细信息，请参阅 [GDL 宏](gdl-macros.md)。
