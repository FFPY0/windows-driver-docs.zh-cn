# <a name="hp-gl2-supported-fonts"></a>HP-GL/2 支持的字体





HP-GL/2 支持与为光栅模式定义的字体资源相同的字体资源。 矢量和光栅模式均可使用的设备字体将添加到 GPD 和资源文件中。
