# <a name="layout"></a>布局


架构路径： \\ 打印机布局

节点类型：属性

Layout 属性包含有关打印作业的每一页的外观的信息。 这包括可用的媒体类型以及如何放置有关每个工作表的信息。

Layout 属性是以下属性的父项：

[NumberUp](numberup2.md)

[方向](orientation2.md)
