# <a name="miscellaneous-gdl-directives"></a>其他 GDL 指令


GDL 提供了两个其他指令：

-   \*IgnoreBlock 导致 GDL 分析器忽略构造的值和内容。

-   \*% (星号加上百分号) 导致分析器忽略行的其余部分。 您可以使用此指令添加注释。
