# <a name="numeric-values"></a>数值





指定为 GPD 文件中的入口值或参数值的所有数值都必须是整数。 不允许使用小数点，但文本字符串中除外。

除非前面有一个减号，否则数值将被假定为正值。

数值被假定为十进制值，除非前面有0x，在这种情况下，它们是无符号的十六进制值。

\*如果适用于特定 GPD 文件条目的上下文，则可以使用星号字符 () 来指示无限值或 "不在意" 值。
