# <a name="range-template-data-type-issues"></a>范围模板数据类型问题


\*数组大小指令接受一个整数或一个整数范围。  (一个范围是由连字符 (-) 分隔的两个整数，可选择性地括在括号或方括号 \[ 中，如 \* 数组大小： \[ 1-15 \] \]) 。 您可以使用 GPD 通配符 (\*) 来表示任何整数值。 例如，若要定义任意大小的数组，可以使用 \* 数组大小： \* 。
