# <a name="rectangles"></a>矩形





若要在 GPD 文件中描述矩形，请使用 RECT 关键字。 格式为：

**RECT** ( *左* 、 *上* 、 *右* 、 *下* ) 

其中，"*上* *"、"* 上"、"*右*" 和 "*下*" 表示 [数字值](numeric-values.md)，以 [主单位](master-units.md)表示。
