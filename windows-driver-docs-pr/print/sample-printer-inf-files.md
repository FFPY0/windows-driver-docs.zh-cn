# <a name="sample-printer-inf-files"></a>打印机 INF 文件示例





示例打印机 INF 文件随 Windows 驱动程序工具包一起提供， (WDK) 位于 \\ src \\ print \\ inf \\ driver5 子目录中。 此文件是一个模板，用于说明用于安装 Unidrv 和 Pscript 微型驱动程序的 INF 文件的布局。 将占位符文件名替换为驱动程序文件的名称。
