# <a name="clearallpathhealthcounters-function"></a>ClearAllPathHealthCounters 函数


ClearAllPathsHealthCounters 方法用于清除 MPIO 管理的每个路径的所有收集到的 MPIO 运行状况统计信息。

<a name="syntax"></a>语法
------

```ManagedCPlusPlus
unknown ClearAllPathHealthCounters(void);
```

<a name="parameters"></a>参数
----------

此函数没有参数。

<a name="return-value"></a>返回值
------------

无

<a name="remarks"></a>备注
-------

此 WMI 方法属于 MPIO \_ wmi \_ 方法 wmi 类。
