# <a name="interface-description"></a>接口说明


故障转储筛选器驱动程序应通过在注册表中输入驱动程序密钥来注册故障转储驱动程序。 崩溃转储筛选器驱动程序还应实现在 [筛选器回调例程](filter-callback-routines.md) 主题中定义的回调例程。
