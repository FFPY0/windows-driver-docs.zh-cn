# <a name="setinitiatornodename"></a>SetInitiatorNodeName


**SetInitiatorNodeName** 方法通知发起方发起方名称已更改。

此 WMI 方法属于未发布的 [MSiSCSI \_ 操作 WMI 类](msiscsi-operations-wmi-class.md)。 有关 **SetInitiatorNodeName** 方法的参数的说明，请参阅 [**SetInitiatorNodeName \_ IN**](/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_setinitiatornodename_in) 和 [**SetInitiatorNodeName \_ OUT**](/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_setinitiatornodename_out) 结构的成员说明。

实现 MSiSCSI 操作 WMI 类的微型端口驱动程序 \_ 不需要支持 **SetInitiatorNodeName**。
