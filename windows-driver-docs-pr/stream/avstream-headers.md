# <a name="avstream-headers"></a>AVStream 标头





所有内核流式处理和 AVStream 参考资料都涵盖在 Ks 和 Ksmedia 中声明的结构和函数。 这些标头是微型驱动程序为了访问 Microsoft 提供的 KS 和 AVStream 类驱动程序支持而必须包含的标头。
