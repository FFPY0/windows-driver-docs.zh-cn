以下列表描述了前面的表中显示的值的含义：

-   值0x0 到 0x1E (30 decimal) 表示特定的插件号。

-   值0xFF 表示任意可用的子源或目标的插入地址。

-   "Self" 包含 AVCCONNECTINFO 结构所设置的 pin。

-   为源和目标子源插入 (的 **DeviceID** 列中的值) 用于搜索目标 AV/c 设备的物理设备对象 (PDO) ，以向发出 AV/c CCM 命令。
