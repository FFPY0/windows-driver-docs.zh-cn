### <a name="comments"></a>注释

有关如何使用此属性的示例，请参阅 [编码器代码示例](./encoder-code-examples.md)。

微型驱动程序是在属性项中提供静态 **PropertyItem** 说明或处理基本支持查询并填充中的值所必需的。 微型驱动程序还必须指定此属性的默认值。
