# <a name="ksevent_connection_datadiscontinuity"></a>KSEVENT \_ 连接 \_ DATADISCONTINUITY


**KSEVENT \_ CONNECTION \_ DATADISCONTINUITY** 事件允许客户端接收数据中断通知。 当通过 pin 上的流接口发送数据不连续性标志时，将发生此事件。 此事件没有参数。
