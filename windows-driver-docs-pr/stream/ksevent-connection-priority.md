# <a name="ksevent_connection_priority"></a>KSEVENT \_ 连接 \_ 优先级


使用 **KSEVENT \_ 连接 \_ 优先级** 事件，客户端可以接收优先级失败的通知。 当资源由于优先级别而丢失时，将发生此事件。 此事件没有参数。
