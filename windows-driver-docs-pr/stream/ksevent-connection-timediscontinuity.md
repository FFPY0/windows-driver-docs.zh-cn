# <a name="ksevent_connection_timediscontinuity"></a>KSEVENT \_ 连接 \_ TIMEDISCONTINUITY


**KSEVENT \_ CONNECTION \_ TIMEDISCONTINUITY** 事件允许客户端接收时间中断通知。 当通过 pin 上的流接口发送时间不连续性标志时，将发生此事件。 此事件没有参数。
