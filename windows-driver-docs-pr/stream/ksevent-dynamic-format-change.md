# <a name="ksevent_dynamic_format_change"></a>KSEVENT \_ 动态 \_ 格式 \_ 更改


客户端启用 KSEVENT \_ 动态 \_ 格式 \_ 更改事件，以指示发生了动态格式更改。

<a name="remarks"></a>备注
-------

AVStream 硬件编解码器驱动程序生成 KSEVENT \_ 动态 \_ 格式 \_ 更改。 有关详细信息，请参阅 [支持 AVStream 编解码器中的动态格式更改](./supporting-dynamic-format-changes-in-avstream-codecs.md)。
