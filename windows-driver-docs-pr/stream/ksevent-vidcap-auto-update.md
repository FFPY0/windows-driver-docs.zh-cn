<a name="remarks"></a>备注
-------

如果用户在设备上翻转交换机，并更改属性值，则客户端可能会注册此事件以获得通知。 要使此事件可用，硬件实现必须提供对此功能的支持。

有关 DirectShow 筛选器和 KsProxy 的详细信息，请参阅 [内核流式处理代理](/windows-hardware/drivers/ddi/_stream/index)。
