[**KSEVENT \_ 连接 \_ TIMEDISCONTINUITY**](ksevent-connection-timediscontinuity.md)****

[**KSEVENT \_ 连接 \_ 优先级**](ksevent-connection-priority.md)****

[**KSEVENT \_ 连接 \_ ENDOFSTREAM**](ksevent-connection-endofstream.md)****
