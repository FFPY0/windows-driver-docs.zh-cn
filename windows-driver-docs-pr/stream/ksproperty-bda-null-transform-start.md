# <a name="ksproperty_bda_null_transform_start"></a>KSPROPERTY \_ BDA \_ NULL \_ 转换 \_ 启动


客户端使用 KSPROPERTY \_ BDA \_ NULL \_ 转换 \_ 启动来禁用节点之前应用到信号的任何转换，并允许信号无变化地通过节点。
