<a name="remarks"></a>备注
-------

KSP **NodeId** \_ 节点的节点1指定了控制节点的标识符，或设置为−1以指定 pin。

返回的32位值是 [**BDA \_ LockType**](/windows-hardware/drivers/ddi/bdamedia/ne-bdamedia-_bdalocktype)类型的值的按位 "或"，用于指示驱动程序支持的锁类型。

RF 调谐器节点应提供此指示。
