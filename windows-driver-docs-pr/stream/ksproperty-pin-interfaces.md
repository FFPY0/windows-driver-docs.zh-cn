<a name="remarks"></a>备注
-------

\_ \_ 使用 KSP PIN 指定 KSPROPERTY pin 接口 \_ ，其中 **PinId** 成员指定要为其返回可用接口的 pin 工厂。

此属性返回按类驱动程序首选项排序的接口。

Stream 微型驱动程序不需要直接处理此属性;流类驱动程序使用流请求块处理此属性以查询详细信息。
