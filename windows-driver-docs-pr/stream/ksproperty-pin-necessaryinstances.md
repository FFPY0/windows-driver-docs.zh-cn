<a name="remarks"></a>备注
-------

使用 KSP pin 指定此属性 \_ ，其中成员指定相关的 PIN 工厂。

KSPROPERTY \_ pin \_ NECESSARYINSTANCES 返回类型为 ULONG 的值，指定 pin 工厂必须实例化的最小 pin 数。

类驱动程序不处理此属性;stream 微型驱动程序必须自行提供处理。
