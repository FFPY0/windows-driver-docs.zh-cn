# <a name="ksproperty_preferred_capture_surface"></a>KSPROPERTY \_ 首选 \_ 捕获 \_ 图面


KSPROPERTY \_ 首选 \_ 捕获 \_ SURFACE 属性返回捕获驱动程序的首选内存目标以便捕获，无论是 VRAM 还是系统内存类型。

若要使用 VRAM 传输，捕获微型驱动程序必须支持此属性。
