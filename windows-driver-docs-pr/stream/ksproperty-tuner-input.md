# <a name="ksproperty_tuner_input"></a>KSPROPERTY \_ 调谐器 \_ 输入


KSPROPERTY \_ 调谐器 \_ 输入属性描述当前调谐模式下的调谐器输入，例如，在电缆和天线调谐器输入之间进行选择。 必须实现此属性。
