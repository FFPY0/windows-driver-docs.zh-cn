# <a name="ksproperty_tuner_standard_mode"></a>KSPROPERTY \_ 调谐器 \_ 标准 \_ 模式


KSPROPERTY \_ 调谐器 \_ 标准 \_ 模式属性检索一个布尔值，该值指示驱动程序是否可以将优化设备设置为自动检测信号本身的调谐器标准。 此属性可以选择实现。
