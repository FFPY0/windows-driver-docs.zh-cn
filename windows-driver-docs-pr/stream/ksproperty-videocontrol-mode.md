# <a name="ksproperty_videocontrol_mode"></a>KSPROPERTY \_ VIDEOCONTROL \_ 模式


KSPROPERTY \_ VIDEOCONTROL \_ mode 属性控制映像生产的模式。 这包括用于水平和垂直图像翻转的设置，以及通过外部触发器启用帧捕获。 此属性是可选的。
