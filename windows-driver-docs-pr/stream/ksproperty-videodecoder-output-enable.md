# <a name="ksproperty_videodecoder_output_enable"></a>KSPROPERTY \_ VIDEODECODER \_ OUTPUT \_ ENABLE


KSPROPERTY \_ VIDEODECODER \_ output \_ ENABLE 属性控制位于共享视频端口总线上的视频解码器的三状态输出。 此属性是可选的。
