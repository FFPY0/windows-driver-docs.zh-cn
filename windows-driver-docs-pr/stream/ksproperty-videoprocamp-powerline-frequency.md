# <a name="ksproperty_videoprocamp_powerline_frequency"></a>KSPROPERTY \_ VIDEOPROCAMP \_ POWERLINE \_ FREQUENCY


KSPROPERTY \_ VIDEOPROCAMP \_ POWERLINE \_ frequency 属性指定本地电源线频率。 如果设备支持荧光灯环境的抗闪烁处理，则可能需要此频率。
