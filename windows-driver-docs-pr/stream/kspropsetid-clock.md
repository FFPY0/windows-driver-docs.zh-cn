[**KSPROPERTY \_ 时钟 \_ CORRELATEDPHYSICALTIME**](ksproperty-clock-correlatedphysicaltime.md)

[**KSPROPERTY \_ 时钟 \_ CORRELATEDTIME**](ksproperty-clock-correlatedtime.md)

[**KSPROPERTY \_ 时钟 \_ FUNCTIONTABLE**](ksproperty-clock-functiontable.md)

[**KSPROPERTY \_ 时钟 \_ PHYSICALTIME**](ksproperty-clock-physicaltime.md)

[**KSPROPERTY \_ 时钟 \_ 解析**](ksproperty-clock-resolution.md)

[**KSPROPERTY \_ 时钟 \_ 状态**](ksproperty-clock-state.md)

[**KSPROPERTY \_ 时钟 \_ 时间**](ksproperty-clock-time.md)
