[**KSPROPERTY \_ 波形 \_ 输出 \_ 功能**](ksproperty-wave-output-capabilities.md)

[**KSPROPERTY \_ 波形 \_ 频率**](ksproperty-wave-frequency.md)

[**KSPROPERTY \_ 波形 \_ 音量**](ksproperty-wave-volume.md)
