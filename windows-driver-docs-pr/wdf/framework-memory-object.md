# <a name="framework-memory-object"></a>框架内存对象


[!include[UMDF 1 Deprecation](../includes/umdf-1-deprecation.md)]

通过 [IWDFMemory](/windows-hardware/drivers/ddi/wudfddi/nn-wudfddi-iwdfmemory) 接口向驱动程序公开框架内存对象。 它提供对内存块的访问。
